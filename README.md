# 💖 Birthday Surprise Website — Setup Guide

## Folder Structure (Very Important!)

```
your-repo/
│
├── index.html          ← The main file (already done ✅)
│
└── assets/             ← CREATE this folder and put all your files here
    ├── image1.jpg
    ├── image2.jpg
    ├── image3.mp4
    ├── image4.jpg
    ├── image5.png
    ├── ...and so on up to image50 (or however many you have)
```

## File Naming Rules
- **Images**: `image1.jpg`, `image2.png`, `image3.jpeg`, `image4.webp`
- **Videos**: `image5.mp4`, `image6.mov`, `image7.webm`
- Name them in **number order**: image1, image2, image3... up to your last one
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.mp4`, `.mov`, `.webm`

## Deploy to GitHub Pages

1. Create a new **GitHub repository** (make it Public)
2. Upload `index.html` and the `assets/` folder
3. Go to **Settings → Pages**
4. Under "Branch", select `main` → `/ (root)` → click **Save**
5. Wait ~1 minute, your site will be live at:
   `https://YOUR-USERNAME.github.io/REPO-NAME`
